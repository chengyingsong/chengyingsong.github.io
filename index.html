C语言

发表于 2019-07-13 | 分类于 C

第一章（了解）

进制转换

  1. 由二进制转换为10进制，二进制每一位乘以乘以权值。第n位的权值为2的n-1次方，小数点后第n位为2的-n次方。二进制运算逢2进1，规则与十进制类似
  2. 十进制转换为2进制，一直除2，直到值为1，到着取余数，即是2进制值。
  3. 二进制与16进制和8进制一般有对应关系，即4位分组法或者3位分组法。

如111，代表8进制的7，而1111，代表16进制的f。

  • 8进制和16进制转换为10进制和二进制转换类似，也是乘以权值，只是底为8和16。小数部分类似。
  • 10进制转化为8或者16进制，也类似，就是除8或者16取余。

git

发表于 2019-07-13 | 分类于 git

工作区——-git add ———暂缓区———-git commit - m ———-版本库（本地）（主分支）

命令

  • pwd 显示版本库路径
  • git add “file “ 把文件添加到仓库（可反复使用，或者同时添加多个文件）
  • git commit -m < message > 完成添加文件
  • git status 查看修改结果（掌握仓库状态）
  • git diff 对比工作区和暂缓区的不同
  • git log 显示提交日志
  • git log —pretty=oneline 显示版本号提交日志
  • git中用HEAD表示当前版本，也就是最新提交的版本（提交ID）
  • git reset —hard 版本号 返回版本
  • cat 文件 查看文件内容
  • git reflog 记录命令及版本号
  • git checkout — file 丢弃工作区的修改【==—== 很重要，没有就变成了切换到另一个分支】
  • git reset HEAD file 把暂存区的修改撤销掉，即已经add后，先使用这个命令把暂缓区修改撤销再丢弃工作区修i该。

如果已经commit到版本库，使用git reset —hard 版本号 命令返回原来版本以撤销版本库修改

删除文件，首先使用rm file命令手动删除工作区文件，然后使用git rm file，删除版本库文件并git commit。如果在删除工作区文件之后想要恢复文件，可以使用git checkout — file命令撤销。

python笔记

发表于 2019-07-13 | 分类于 python

特殊点

  1. print函数是将其中的所有值都计算完毕再开始输出。即print(1,2,1/0)。由于计算途中出现错误是不会有输出的。
  2. python中没有静态局部变量，但是可以使用闭包，默认参数或者列表对象以及nonlocal关键字实现
  3. python的@语法。当解释器读到@的这样的修饰符之后，会先解析@后的内容，直接就把@下一行的函数或者类作为@后边的函数的参数，然后将返回值赋值给下一行修饰的函数对象。

变量

变量名第一个字符必须是字母或下划线，不能是数字，也不能是python的关键字，而且英文大小字母敏感。

==python的变量不需要声明，变量的赋值操作即是变量声明和定义的过程。还可以给几个变量同时赋值。如，a,b,c=1,2,3==

python库的使用

发表于 2019-07-12 | 分类于 python

turtle库

  • 引入模块 import turtle
  • turtle.setup(a,b,c,d) a,b决定绘制界面的长和宽，，c,d决定绘图界面出现在屏幕的位置，若无设置默认为出现在屏幕中央。

turtle坐标体系

绝对坐标，在绘图界面中的坐标系的坐标

  • turtle.goto() 线性绝对坐标的移动

空间坐标（海龟坐标），以海龟的视角，前后左右方向的坐标

  • turtle.bk(d) 海龟后退d单位
  • turtle.fd(d) 海龟前进d单位
  • turtle.circle(r,angle) 海龟向左前方以半径r移动angle角度

绝对角度坐标 以角度为坐标

  • turtle.seth(angle) 改变海龟前进的方向角度 （有负角）
  • turtle.left（angle) 海龟向左移动angle角
  • turtle.right(angle) 海龟向右移动angle角
